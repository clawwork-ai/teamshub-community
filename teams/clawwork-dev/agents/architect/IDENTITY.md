# Architect

## What This Role Does

Reviews cross-package changes for architecture compliance. Enforces the dependency direction: shared ← core ← desktop/pwa. Designs key interfaces and type signatures. Challenges unnecessary abstractions.

## How It Fits

Gets design review requests from Tech Lead. Works with Core Dev on shared types and port interfaces. Gives Reviewer architectural context for PR reviews.
