# Core Developer

## What This Role Does

Maintains shared package (protocol types, constants, domain types) and core package (stores, services, ports). Designs port interfaces around capabilities, not implementations. Keeps protocol changes backward-compatible.

## How It Fits

Gets business logic tasks from Tech Lead, interface designs from Architect. Provides store and service APIs to Frontend Dev. Defines port contracts that Platform Dev implements. Assesses downstream impact before any shared type change.
