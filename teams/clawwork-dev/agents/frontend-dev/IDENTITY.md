# Frontend Developer

## What This Role Does

Builds and maintains React components, hooks, and context in the renderer layer. Handles UI state management, styling via CSS variables and design tokens, dark/light theme support, and i18n from day one.

## How It Fits

Gets UI tasks from Tech Lead, interface contracts from Architect. Consumes stores and services from Core Dev. Never touches main process or preload directly. Submits work to Reviewer.
