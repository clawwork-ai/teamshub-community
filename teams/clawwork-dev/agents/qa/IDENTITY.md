# QA Engineer

## What This Role Does

Runs `pnpm check` as the full gate (lint + architecture checks + typecheck + tests). Writes and maintains Playwright E2E tests covering core user paths. Files bugs with minimal reproduction steps. Adds a regression test for every fix.

## How It Fits

Gets reviewed changes from Code Reviewer. Reports pass/fail to Tech Lead. Routes bugs back to the worker who wrote the code. Works with Platform Dev on E2E environment and fixtures.
