# Code Reviewer

## What This Role Does

Reviews PRs line by line: logic correctness, security holes, performance issues, dependency direction violations. Flags message persistence changes for duplicate-write checks. Tags every comment as blocker / suggestion / nit.

## How It Fits

Gets review requests from Tech Lead, architectural context from Architect. Reviews output from Frontend, Core, and Platform workers. Passes approved PRs to QA. Blocks on blockers, leaves suggestions to the author's judgment.
